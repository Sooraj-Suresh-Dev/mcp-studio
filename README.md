## ✨ Overview

**MCP Studio** is a state-of-the-art platform designed to simplify the development, testing, and deployment of MCP (Model Context Protocol) servers. Built with a focus on **Premium Aesthetics** and **Seamless UX**, it provides developers with everything they need to extend AI capabilities.

### 🚀 Key Features

*   **⚡ MCP Server Generator**: Instantly scaffold production-ready MCP servers with custom tools and resources.
*   **🩺 MCP Validator**: Real-time schema validation and tool simulation to ensure your server meets standard specifications.
*   **📚 Learning Hub**: Comprehensive guides, "What is MCP?" articles, and interactive tutorials.
*   **🖼️ Templates Gallery**: Downloadable configurations for popular servers like SQLite, GitHub, and Brave Search.
*   **🔗 Connection Wizard**: Generate client-side configurations for Claude Desktop and other MCP hosts.
*   **📱 Fully Responsive**: A beautiful experience across all devices, from desktop to mobile.

## 🏗️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/mcp-toolkit.git
   cd mcp-toolkit
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Launch the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

MCP Studio is built with a **"Dark Mode First"** aesthetic, utilizing:
- **Glassmorphism**: Layered transparency and background blurs.
- **Dynamic Gradients**: Vibrant accent colors (`#6c5ce7` to `#a78bfa`).
- **Smooth Animations**: Carefully timed transitions and scroll-triggered effects.
- **Premium Typography**: Using Inter and modern sans-serif fonts for maximum readability.
